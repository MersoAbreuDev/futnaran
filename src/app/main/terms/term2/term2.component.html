Termo de Participação

Este termo define quem pode participar do campeonato e os requisitos que devem ser atendidos. Por favor, leia atentamente os critérios a seguir:

  • Apenas convidados: A participação é restrita a jogadores que foram formalmente convidados para o campeonato.
  • Não é transferível: A vaga no campeonato não pode ser transferida para outra pessoa. Somente o jogador originalmente convidado pode participar.
  • Estado de saúde: Todos os jogadores devem estar em bom estado de saúde e serem capazes de participar do campeonato de forma segura.
  • Residência: O jogador deve estar registrado no município onde o campeonato está sendo realizado.
  • Pagamento das taxas: A participação só será confirmada após a realização do pagamento de todas as taxas impostas. As taxas devem ser pagas dentro dos prazos estabelecidos.
  • Datas limites: Todos os pagamentos devem ser realizados até as datas limites estipuladas. Após o prazo, o jogador não poderá participar do campeonato.

Certifique-se de cumprir todos os requisitos para garantir sua participação no campeonato. Caso tenha alguma dúvida, entre em contato com a organização.
